Tables
- Users
    - Username (primary key)
    - First Name
    - Last Name
    - Followers (foreign key)
    - Artists (foreign key)

- Artists
    - id (primary key - taken from api)
    - Genre (taken from api)
    - Username (foreign key)



- Followers
    - id (primary key)
    - user (foreign key)
    - follower (foreign key)
